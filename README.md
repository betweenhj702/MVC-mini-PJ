# 아몬드 쇼핑몰

For six days of this Semi Project,  I practiced how to code an Website  and learned stream of data,  using MVC model form.

## 개요
- 시연영상 : https://www.youtube.com/watch?v=QN_Db7tPuak

- 사용 기술
